🚀 Front-end study project developed at DIO
This is an educational project, created to practice HTML and CSS, reproducing only layout and colors inspired by streaming platforms like HBO.
No logos, images, or official brand content are used, and it is not commercial.

📖 Sobre o projeto / About the project

NexaPlay é uma aplicação web de estudo que simula a interface de uma plataforma de streaming. O objetivo principal é consolidar conhecimentos em front-end, aplicando boas práticas de desenvolvimento e design.

NexaPlay is a study web application that simulates a streaming platform interface. The main goal is to strengthen front-end skills, applying best practices in development and design.
